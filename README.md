# 🌍 Global Trivia Map Explorer

An interactive geography learning game that combines **world exploration**, **country facts**, and **trivia challenges** into a fun educational experience. Discover countries, test your knowledge, earn XP, and master regions across the globe.

---

## ✨ Features

### 🌎 Interactive Country Exploration

- Explore countries around the world
- Learn key facts about each nation
- Search countries by name
- Filter countries by region

### 🎮 Multiple Game Modes

| Mode | Description |
|--------|-------------|
| 🧭 **Explore** | Browse countries, learn facts, then start trivia when ready |
| 🎯 **Challenge** | Jump directly into trivia challenges |
| 📖 **Learn** | Study country information before taking quizzes |
| ⭐ **Daily Quest** | Receive a random unmastered country challenge |

---

## 🚀 How to Play

### 1️⃣ Choose a Game Mode

#### 🧭 Explore Mode
Browse countries freely, read facts, and choose when to begin trivia challenges.

#### 🎯 Challenge Mode
Select a country and immediately start the quiz.

#### 📖 Learn Mode
Review the country's information card before beginning the challenge.

#### ⭐ Daily Quest
Jump into a random unmastered country's trivia challenge.

---

### 2️⃣ Explore a Country

Each country includes:

- 🏛️ Capital City
- 👥 Population
- 🗣️ Official Language(s)
- 💰 Currency
- 🎉 Fun Fact

---

### 3️⃣ Complete Trivia Challenges

Each country contains:

- ❓ 3 Multiple Choice Questions
- 🌍 Geography, culture, and history topics
- ⚡ Instant feedback after every answer

#### Answer Results

- ✅ **Green** = Correct
- ❌ **Red** = Incorrect

Regardless of your answer, the correct fact is revealed so learning continues throughout the game.

---

### 4️⃣ Earn XP & Master Countries

#### ⭐ XP Rewards

- ✅ Correct Answer = **+20 XP**

#### 🏆 Mastery System

Every correct answer increases a country's mastery level.

- 📈 Build mastery over time
- 🟢 Reach **100% Mastery** to fully master a country
- 🎯 Mastered countries are highlighted with a green completion card
